0.(Permanently) Add the noetl src to your PYTHONPATH
export PYTHONPATH="${PYTHONPATH}:/your/path/to/noetl/noetl/src"

1. install nosetests for
sudo pip install -upgrade nose

2. find nosetests under /Library/Frameworks/Python.framework/Versions/2.7/bin

3. run tests using
python /Library/Frameworks/Python.framework/Versions/2.7/bin/nosetests testFile.py
